Código del Ejemplo

'''
crear una frase larga con palabras repetidas donde exprese un sentimiento positivo
y sentimientos negativos, analizar la frase completa, eliminando signos de puntuacion
, tokenizando la frase, y recoriendo la frase con if contabilizar cuantas
palabras positivas y cuants negativas se encontraron, y de ahi determinar
si la frase es positiva o negativa
'''

#creando listas con palabras para detectar el sentir de la frase
palabraspositivas=['feliz','contento','maravillosa','bonito','bonita','color']
palabrasnegativas=['triste','solo','aburrido','malo','deprimido','horrible','muerto','demonios']

frase="solo, triste feliz bonito color"

texto_limpio=frase.replace(",","").replace(".","").lower()
palabras = texto_limpio.lower().split()
verificadas=[]
positivos=0
negativos=0

for palabra in palabras:
    if palabra  in palabraspositivas:
        positivos += 1
    if palabra in palabrasnegativas:
        negativos +=1

print("palabras positivos;",positivos)
print("palabras negativas:",negativos)

for palabra in palabras: #python esta dentro de tokens
    if palabra not in verificadas:
        print(palabra,"tiene",len(palabra),"letras")
        verificadas.append(palabra)

if positivos > negativos:
    print("Tu frase es positiva")

elif negativos > positivos:
    print("Tu frase es negativa")

Glosario de Funciones

replace()
Sirve para eliminar signos de puntuación (como comas o puntos).
lower()
Devuelve un conjunto (set) con los tokens únicos del texto, eliminando duplicados. Útil para conocer el vocabulario utilizado sin repeticiones.
split()
Divide una cadena de texto en una lista de palabras (tokens). Permite analizar palabra por palabra.
for ... in ...
Crea un bucle que recorre cada elemento de una lista. Se usa para procesar todas las palabras de la frase.
if ... in ...
Verifica si un elemento está dentro de una lista. Se usa para saber si una palabra es positiva o negativa.
len()
Devuelve la cantidad de caracteres (letras) que tiene una palabra. Se usa para contar cuántas letras tiene cada palabra única.
append()
Agrega un nuevo elemento al final de una lista. En el código se usa para guardar palabras ya verificadas y no repetirlas.
if / elif / else
Estructura condicional que evalúa distintas opciones. En el programa, determina si la frase es positiva, negativa o neutra.
