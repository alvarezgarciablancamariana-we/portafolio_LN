Tema 03

Contar palabras

Código del Ejemplo

#frase ejemplo
frase= "python es divertido y python poderoso "

#1.-limpiar el texto (minuscula y sin puntuacion)
frase_limpia= frase.replace(".","").replace(".","").lower()

#2.- Dividir en palabras (tokenizacion)
tokens=frase_limpia.split()
#3.- contar palabras usando diccionario
frecuencia={}
for palabra in tokens:
    if palabra in frecuencia:
        frecuencia[palabra]+=1
    else:
        frecuencia[palabra]=1

#4.-mostrar resultados
print(frecuencia)

Glosario de Funciones

replace()
Reemplaza un carácter o cadena por otro dentro del texto (por ejemplo, eliminar signos de puntuación).
lower()
Convierte todo el texto a letras minúsculas para uniformar las comparaciones.
frecuencia = {}
Crea un diccionario vacío donde se almacenarán las palabras como claves y la cantidad de veces que aparecen como valores.
for palabra in tokens:
Recorre cada palabra de la lista tokens para contar su frecuencia.
if palabra in frecuencia:
Verifica si la palabra ya existe como clave en el diccionario
frecuencia[palabra] += 1
Aumenta el contador de la palabra si ya se encontraba registrada.
frecuencia[palabra] = 1
Agrega una nueva palabra al diccionario con un conteo inicial de 1.
