    for x in componentes:
        if x["nombre"].lower() == componente:
            print(f"\nConcepto de '{x['nombre']}':")

Glosario de Funciones

print()
Muestra texto en la pantalla de la consola.
for
Recorre todos los elementos de la lista componentes para imprimirlos o compararlos.
while True:
Crea un ciclo infinito para que el programa siga pidiendo un componente hasta que se escriba uno correcto.
input()
Permite que el usuario escriba el nombre del componente que quiere aprender.
.lower()
Convierte el texto a minúsculas para comparar sin importar cómo lo escribió el usuario.
if
Verifica si el componente escrito coincide con algún elemento de la lista.
break
Sale del ciclo cuando el componente fue encontrado correctamente.
else
Se ejecuta cuando el componente no coincide con ninguno; muestra error y pide reintentar.
componentes = [...]
Es la lista que contiene todos los componentes y sus conceptos en formato de diccionario.
"nombre" / "concepto"
Llaves dentro del diccionario que almacenan el nombre del componente y su explicación.
